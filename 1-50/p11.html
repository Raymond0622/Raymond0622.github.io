Somewhat of a step-up in difficulty from the previous 10 problems. However, it's pretty straightforward. We have to check four directions and keep track of the maximum sum. The vertical/horiztonal directions seem pretty easy How do we do diagonals?

Take a simpler 4x4 matrix below:

If we want to find, let's say max of three element sums using only diagonal direction, we can iterate through all possible blocks of two elements in each possible "sub-diagonals" For example, we see there are 5 possible diagonals in this 4x4 matrix. Each color is denotes each different diagonal with at least 3 elements.

08 02 22 21

We can see that there exists 3 different diagonals. However, there are two different rules when coming up with a for loop for red and purple line. The main diagonal (green) is simple enough with matrix[i][i]. So, there are two for loops to account for diagonals abaove the main and below. Then, we just take each diagonal and iterate through every possible consecutive 3 element sums! How about diagonals such as 49-24-23?
